# Jobs To Be Done — KnowFlow AI

Job statements

- When I need to find an answer inside our contracts, I want to ask a question and get a concise, sourced answer so I can act quickly.

- When I onboard to a new team, I want to search across operational docs so I can learn procedures without reading everything.

- When auditing decisions, I want to trace back the source documents that support an AI-generated claim.

Acceptance criteria
- Answers must include sources and pointers to document locations.
- Search should prioritize recency and relevance based on metadata filters.

Edge cases
- Poorly formatted PDFs with embedded images (no OCR) — flag for manual processing.
